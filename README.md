# Tema 1
### Să se creeze mai multe comenzi pentru laptop-uri.
### O comandă va avea ca atribut marca: HP, Dell, Lenovo.
### Un laptop va avea componente (itemi) de tip: MEMORY, WEB, HDD, SSD, VIDEO (caracteristică, preț).
Exemplu:
- MEMORY, 8 (Gb), 450 (RON)
- HDD, 1024 (Gb), 250 (RON)
- Web, 1080 (HD), 370 (RON)
- SSD, 500 (Gb), 600 (RON)
- VIDEO, 2 (Gb), 780 (RON).
